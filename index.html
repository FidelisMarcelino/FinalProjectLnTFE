Pudding Fest 2024

Digital Odyssey: Navigating the Future

PuddingFest is an annual event held by PT ChipiChapa, where well-known companies in Indonesia will showcase their latest technologies online. This event offers a series of presentations from each company, giving participants the opportunity to gain insight into the latest innovations in the world of technology and interact with representatives of participating companies.

OUR EVENTS

Start Designing User Friendly Interface Using Figma

This event provides participants with a chance to boost their UI design skills with Figma, exploring effective principles, innovative techniques, and knowledge sharing with professionals. With a focus on project-based learning and dynamic interaction, it aims to inspire and empower attendees in creating impressive interface designs.

TechInsight Summit with Flutter

This is an event that combines various technology insights with a focus on the Flutter development platform. Participants will be invited to understand various perspectives on using Flutter for cross-platform application development, as well as share knowledge and experience with industry experts and professionals. With interactive sessions and in-depth discussions, this event aims to provide deep insight into the potential and best practices in development using Flutter, so that participants can optimize the use of this technology in their projects.

SQL Dive: Unlocking Data Insights

"SQL Dive: Unlocking Data Insights" is an event that invites participants to understand more deeply about the SQL (Structured Query Language) programming language and how to use it to extract insights from data. Through interactive sessions, participants will learn about basic SQL concepts, complex queries, and best practices in analyzing data. This event is aimed at beginners and professionals who want to improve their skills in managing and analyzing data using SQL.
